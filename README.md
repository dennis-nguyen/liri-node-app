Liri-Node-App
===============

### Summary
**Liri-Node-App** was created to practice Node.JS.

### Technologies
- Node.JS
- Spotify Package
- Twitter Package

### CLI Commands
- node liri.js my-tweets
- node liri.js spotify-this-song '<song name here>'
- node liri.js movie-this '<movie name here>'
- node liri.js do-what-it-says
